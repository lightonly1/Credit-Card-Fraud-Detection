# Credit-Card-Fraud-Detection
A Machine Learning project to detect credit card fraud using classification algorithms
This project demonstrates a machine learning-based approach to detect fraudulent credit card transactions using real-world dataset.

## Dataset
The dataset used in this project is available at [Kaggle](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud). It contains transactions made by credit cards in September 2013 by European cardholders.

##  Techniques Used
- Data Preprocessing (null values, scaling)
- Exploratory Data Analysis (EDA)
- Handling Class Imbalance using SMOTE
- Classification Models:
  - Logistic Regression
  - Random Forest
  - XGBoost

##  Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix
- ROC AUC Curve

##  Results
Achieved promising results with balanced precision and recall. The models successfully detect fraudulent activities with high accuracy.
